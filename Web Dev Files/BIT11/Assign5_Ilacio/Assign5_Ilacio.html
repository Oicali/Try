Embed an Audio

          Use the < audio > tag to embed and play a sound content in a document, like music or other audio streams. The controls attribute will add play, pause, and volume as controls. The < source > element allows user to specify audio files. The text between the < audio > and < /audio > tags will only be displayed in browsers that do not support the < audio > format. MP3, WAV, and OGG are supported audio formats in HTML. If you want to play an audio automatically, add the autoplay attribute. Results from the source code above are shown below:

Embed a Video

          The < video > tag is used to embed a video content in an HTML file , such as a movie clip or other video streams. The controls attribute will also add play, pause, and volume as video controls. The width and height are necessary to be included to prevent the webpage to flicker while the video loads. The < source > element also allows user to specify video files. The text between the < video > and < /video > tags will only be displayed in browsers that do not support the < video > element. MP4, Webm, and OGG are the 3 supported video formats in HTML. If you want to play the video automatically, add the autoplay attribute. However, there are browsers that do not allow you to play the video automatically unless you add the muted attribute for security purposes. Results from the source code above are shown below:

Create a Collapsible Text

          To create a collapsible text, the tags < details > and < summary > are required. Additional details are specified in the < details > tag, which the user can open and close as needed. In order to create an interactive widget that the user can open and close, the < details > tag is frequently utilized. The widget is closed by default. When it is opened, it expands to reveal the contents. Meanwhile, the < summary > tag defines a visible heading for the < details > element. The heading can be clicked to view/hide the details.
